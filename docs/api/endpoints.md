# API Documentation

## Overview
The backend exposes a REST API for managing users, properties, bookings, payments, and reviews.

## Endpoints
- **Users**
  - `POST /api/users/register`
  - `POST /api/users/login`
- **Properties**
  - `GET /api/properties`
  - `POST /api/properties`
- **Bookings**
  - `POST /api/bookings`
  - `GET /api/bookings/{id}`
- **Payments**
  - `POST /api/payments`
- **Reviews**
  - `POST /api/reviews`
